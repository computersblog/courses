Lesson 1: What is HTML?

HTML stands for HyperText Markup Language. It is the language used to build the structure of websites.

HTML uses tags. A tag tells the browser what something is.

Example:
<h1>Hello</h1>

<h1> means a big heading Hello is the text </h1> closes the tag
